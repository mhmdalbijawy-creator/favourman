# Favourman International - Executive Corporate Website

A luxury, executive-tier corporate website built for **Favourman International**, a global leader in commercial services and strategic consultancy.

## 🌟 Vision & Design
The website follows a high-end, minimalist aesthetic:
- **Luxurious Theme**: Single dark mode with glowing gold (#D4AF37) accents.
- **Brand Identity**: Featuring a custom-designed abstract Phoenix logo representing resilience and global reach.
- **Interactive Experience**: Built with Framer Motion for smooth, premium animations and micro-interactions.
- **Global Reach**: Interactive world map showcasing strategic presence in 200 countries, with core focuses in Egypt, India, Pakistan, and Sudan.

## 🚀 Key Features
- **21 Strategic Capabilities**: Detailed showcase of the company's vast service portfolio (Oil & Gas, Mining, Military Logistics, etc.).
- **Multilingual Support**: Fully localized in English and Arabic with seamless switching.
- **Dynamic Service Architecture**: Professional service detail pages with integrated inquiry protocols.
- **Official Documentation**: Dedicated section for institutional transparency and verification.
- **Secure Communication**: Integrated contact protocols via Web3Forms with intelligent lead tracking.

## 🛠️ Technology Stack
- **Frontend**: React.js 18 + Vite
- **Styling**: Tailwind CSS (Executive custom theme)
- **Animations**: Framer Motion
- **Internationalization**: i18next
- **Routing**: React Router DOM v7
- **Form Handling**: Web3Forms API

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/website.mo.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```
